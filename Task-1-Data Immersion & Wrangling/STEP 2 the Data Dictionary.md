# the Data Dictionary
| Column Name                 | Data Type   | Description                           | Example                |
| --------------------------- | ----------- | ------------------------------------- | ---------------------- |
| gender                      | Categorical | Student gender                        | male, female           |
| race/ethnicity              | Categorical | Student race/ethnicity group          | group A–E              |
| parental level of education | Categorical | Highest education level of parent(s)  | bachelor’s degree      |
| lunch                       | Categorical | Lunch type indicating economic status | standard, free/reduced |
| test preparation course     | Categorical | Whether student completed test prep   | none, completed        |
| math score                  | Integer     | Math exam score (0–100)               | 72                     |
| reading score               | Integer     | Reading exam score (0–100)            | 90                     |
| writing score               | Integer     | Writing exam score (0–100)            | 88                     |
